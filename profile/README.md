# stigmaRverse

**An open-source ecosystem for measuring and analyzing structural stigma.**

`stigmaRverse` is a collection of tools developed by researchers at the [Ecologies of Liberation Lab](https://github.com/) at the University of Illinois Urbana-Champaign to support research on structural stigma, minority stress, and LGBTQ+ health.

## Repositories

| Repo | Description |
|------|-------------|
| [`stigmaR`](https://github.com/stigmaRverse/stigmaR) | Core R package: indices, models, and functions for structural stigma measurement |
| [`stigmaRdata`](https://github.com/stigmaRverse/stigmaRdata) | Data processing pipeline |
| [`stigmaR.app`](https://github.com/stigmaRverse/stigmaR.app) | Shiny web application for interactive data exploration |

## Getting Started

```r
# Install the core package
remotes::install_github("stigmaRverse/stigmaR")
```
